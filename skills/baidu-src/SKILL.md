# 百度 SRC (BSRC)

## 硬规则（违反即封号）
1. 仅测 *.baidu.com 明确收录范围
2. 禁止全端口/C段/高频扫描（--min-rate ≤ 200）
3. 数据获取 ≤ 10 条，仅证明漏洞存在
4. 禁止内网渗透、禁止拖库
5. 禁止修改/删除数据，存储XSS用 console.log
6. 不注册账号（除非需要且仅限2个测试号）
7. 发现漏洞立即停止深入，不扩大影响

## 业务系数
```
高系数: 搜索/地图/网盘         → 赏金最高
中系数: 贴吧/Apollo/车联网
低系数: 其他边缘业务
```

## 高回报策略
```
1. SSRF 打内部服务（百度云/Apollo/内部API）
2. 越权访问用户数据（订单/个人信息）
3. APP 逻辑漏洞（支付/优惠券）
4. 小程序/轻应用未授权
5. 百度云(AKI)配置缺陷
```
